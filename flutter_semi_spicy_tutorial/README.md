하루 빨리 위젯을 분리하고, 최적의 폴더구조를 설계하여 boiler-plate folder structure를 제작하고 싶다.
이젠 실제 애플리케이션 클론코딩 과정을 통해서 더욱 단단한 플러터를 학습할 것이다.
